# 🚀 Automated Setup Guide\n\n## ⚡ One-Click Setup\n\nYour Haven e-commerce website includes **automated setup scripts** for both Mac/Linux and Windows!\n\n---\n\n## 🖥️ **FOR WINDOWS USERS**\n\n### Method 1: Using Batch Script (Easiest)\n\n1. **Open File Explorer**\n2. **Navigate to**: `haven-ecommerce/frontend/`\n3. **Double-click**: `setup.bat`\n4. **Wait for installation** (takes 2-3 minutes)\n5. **Run**: `start.bat` to start the server\n6. **Open browser**: http://localhost:3000\n\n### Method 2: Using Command Prompt\n\n```cmd\ncd frontend\nsetup.bat\n```\n\nThen:\n\n```cmd\nstart.bat\n```\n\n---\n\n## 🍎 **FOR MAC/LINUX USERS**\n\n### Method 1: Using Terminal\n\n```bash\ncd frontend\nchmod +x setup.sh\n./setup.sh\n```\n\nThen:\n\n```bash\nchmod +x start.sh\n./start.sh\n```\n\n### Method 2: Direct Commands\n\n```bash\ncd frontend\nnpm install\nnpm run dev\n```\n\n---\n\n## 📋 **What These Scripts Do**\n\n### `setup.bat` (Windows) / `setup.sh` (Mac/Linux)\n\n✅ Checks if Node.js is installed\n✅ Verifies npm version\n✅ Installs all dependencies\n✅ Removes old files if needed\n✅ Shows completion message\n\n### `start.bat` (Windows) / `start.sh` (Mac/Linux)\n\n✅ Checks dependencies\n✅ Starts development server\n✅ Shows all available URLs\n✅ Auto-opens browser (sometimes)\n\n---\n\n## 🎯 **Quick Start (3 Steps)**\n\n### **WINDOWS:**\n```cmd\ncd frontend\nsetup.bat\nstart.bat\n```\n\n### **MAC/LINUX:**\n```bash\ncd frontend\nchmod +x setup.sh && ./setup.sh\nchmod +x start.sh && ./start.sh\n```\n\n---\n\n## ✅ **Expected Output**\n\nWhen running the server, you should see:\n\n```\n🏠 Haven E-commerce - Development Server\n==========================================\n\n🚀 Starting development server...\n\n📍 Server will run at: http://localhost:3000\n\nPages available:\n  🏠 Home:     http://localhost:3000\n  🛍️  Products: http://localhost:3000/products\n  🛒 Cart:     http://localhost:3000/cart\n  🔐 Login:    http://localhost:3000/login\n  📊 Admin:    http://localhost:3000/admin\n```\n\n---\n\n## 🌐 **Open Your Website**\n\nOnce you see the message above:\n\n👉 **Open your browser and go to:**\n```\nhttp://localhost:3000\n```\n\n---\n\n## 🛠️ **Troubleshooting**\n\n### ❌ Script Won't Run (Windows)\n\n**Solution:**\n1. Right-click `setup.bat`\n2. Select \"Run as Administrator\"\n3. Click \"Yes\" on any prompts\n\n### ❌ Script Won't Run (Mac/Linux)\n\n**Solution:**\n```bash\nchmod +x setup.sh\n./setup.sh\n```\n\n### ❌ Port 3000 Already in Use\n\n**The script will notify you. Use a different port:**\n\n**WINDOWS:**\n```cmd\nnpm run dev -- -p 3001\n```\n\n**MAC/LINUX:**\n```bash\nnpm run dev -- -p 3001\n```\n\nThen visit: http://localhost:3001\n\n### ❌ Dependencies Won't Install\n\n**Solution:**\n```bash\n# Windows\nsetup.bat\n# Follow prompts to reinstall\n\n# Mac/Linux\n./setup.sh\n# Follow prompts to reinstall\n```\n\n---\n\n## 📱 **Test Your Website**\n\nOnce it's running, visit these pages:\n\n| Page | URL | What to See |\n|------|-----|-------------|\n| 🏠 | http://localhost:3000 | Beautiful home page with products |\n| 🛍️ | http://localhost:3000/products | 12+ household products |\n| 🛒 | http://localhost:3000/cart | Shopping cart with calculations |\n| 🔐 | http://localhost:3000/login | Professional login form |\n| 📊 | http://localhost:3000/admin | Admin dashboard with analytics |\n\n---\n\n## 🎨 **Test Responsiveness**\n\n1. **Open DevTools**: Press `F12` or `Ctrl+Shift+I`\n2. **Toggle Device**: Click the phone icon 📱\n3. **Test on different devices**: Tablet, Phone, Desktop\n\n---\n\n## ⏹️ **Stop the Server**\n\n- **Press**: `Ctrl + C` in your terminal/command prompt\n- **Confirm**: `Y` if prompted\n\n---\n\n## 🚀 **You're All Set!**\n\nYour Haven e-commerce website is now:\n- ✅ Installed\n- ✅ Configured  \n- ✅ Running locally\n- ✅ Ready to preview\n\n**Enjoy! 🏠✨**\n
